INTRODUCTION
------------
  Provides the ability to have a tabbed panel-pane that displays mini-panels
  within it.

  * For a full description of the module, visit the project page:
    https://www.drupal.org/project/panels_ajax_tab

  * To submit bug reports and feature suggestions, or to track changes:
    https://www.drupal.org/project/issues/panels_ajax_tab


INSTALLATION
------------
  Install as you would normally install a contributed drupal module.
  @see https://www.drupal.org/documentation/install/modules-themes/modules-7
  for more details.


CONFIGURATION
-------------
  Please visit for details
  https://www.drupal.org/docs/7/modules/panels-ajax-tabs
